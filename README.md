### Optymalisation Methods

This is repository of optymalisation methods for searching of local minima and zero of a function. All of this method were done for Optymalisation classes at University of Lodz.

## List of methods:

1. Secant
2. Bisection
3. Newton-Raphson
4. Nelder Mead
5. Hooke Jeeves
6. Gradient Descent
